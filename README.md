## Turbowin IoT 프로젝트 (Android, Web)

## 목적
- 공장 원자재, 부자재, 반제품, 완제품 생산관리 홈페이지
- 공정 처리시 조합에 따라 자동으로 재료 수량 차감 및 분류 이동

## 담당
- 디자인, 웹퍼블리싱 제외 1인개발
